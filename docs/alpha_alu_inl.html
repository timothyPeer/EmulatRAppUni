This entire group is Alpha integer execution semantics, suitable for:

•Integer Operate instructions

•Conditional moves

•Compare instructions

•Status/flag evaluation

 

These helpers are pure, side-effect free, and ideal for:

•coreLib

•EBox execution

•Grain reuse

•The overload of sext32 is intentionally listed twice — same name, distinct semantic inputs.
