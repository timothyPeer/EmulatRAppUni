PCTX_helpers

 

Function / Helper Name

Inferred Description

decodePCTX_ASN

Extracts the Address Space Number (ASN) field from a raw PCTX value

decodePCTX_ASTER

Extracts the AST Enable (ASTER) field from a raw PCTX value

decodePCTX_ASTRR

Extracts the AST Request (ASTRR) field from a raw PCTX value

decodePCTX_FPE

Determines whether Floating Point Enable (FPE) is set in PCTX

decodePCTX_PPCE

Determines whether Per-Process Performance Counter Enable (PPCE) is set

encodePCTX

Builds a complete PCTX value from individual ASN, ASTER, ASTRR, FPE, and PPCE fields

getPCTX

Returns the raw 64-bit PCTX value for the specified CPU

setPCTX

Writes a raw 64-bit value to the PCTX register for the specified CPU

getPCTX_ASN

Returns the decoded ASN field for the active CPU PCTX

getPCTX_ASTER

Returns the decoded AST Enable field for the active CPU PCTX

getPCTX_ASTRR

Returns the decoded AST Request field for the active CPU PCTX

getPCTX_FPE

Returns whether floating-point execution is enabled for the current process

getPCTX_PPCE

Returns whether per-process performance counters are enabled

pctx_hw_mtpr_write

Implements HW_MTPR semantics for PCTX using field-select bits to update subfields atomically
