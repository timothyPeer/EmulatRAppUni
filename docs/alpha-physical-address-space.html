================================================================================

ROUTING DECISION TREE

================================================================================

 

Physical Address (PA) arrives at GuestMemory

│

├─ 0x0000000000000000 - 0x0000000000010000  → AlphaMemorySystem (PAL)

│

├─ 0x0000000000002000 - 0x0000000000006000  → AlphaMemorySystem (HWRPB)

│

├─ 0x0000000020000000 - 0x0000000020200000  → SRMFirmwareRegion (SRM)

│

├─ 0x0000000080000000 - 0x0000000880000000  → SafeMemory (RAM)

│   └─ SafeMemory → SparseMemoryBacking

│       └─ 64KB page allocation

│

├─ 0x00000000F0000000 - 0x0000000100000000  → MMIOManager (Device CSRs)

│

├─ 0x0000000880000000 - 0x0000000980000000  → MMIOManager (PCI BARs)

│

└─ Everything else                           → AccessViolation

CRITICAL: PCI AUTO-PLACEMENT ALGORITHM

================================================================================

 

INPUT:

  ramBase = 0x0000000080000000  (from config)

  ramSize = 32 GB               (from config)

 

CALCULATION:

  ramEnd = ramBase + ramSize

         = 0x0000000080000000 + 0x0000000800000000

         = 0x0000000880000000  (34 GB)

 

  pciMemBase = (ramEnd + GB - 1) & ~(GB - 1)

             = (0x0000000880000000 + 0x40000000 - 1) & ~0x3FFFFFFF

             = 0x00000008BFFFFFFF & 0xFFFFFFFFC0000000

             = 0x0000000880000000  (already GB-aligned)

 

RESULT:

  PCI window starts at 0x0000000880000000 (34 GB)

  No collision with RAM (ends at 34 GB)

 

VERIFICATION:

  RAM:    [0x80000000, 0x880000000)   ✅ No overlap

  PCI:    [0x880000000, 0x980000000)  ✅ Starts where RAM ends

 

 

================================================================================

CONFIGURATION EXAMPLE

================================================================================

 

[MemoryMap]

# HWRPB - SRM boot parameters

HwrpbBase=0x2000

HwrpbSize=0x4000

 

# PAL - Architecture library

PalBase=0x0

PalSize=0x10000

 

# SRM - Firmware ROM

SrmBase=0x20000000

SrmSize=0x200000

 

# RAM - Main memory (size from [System])

RamBase=0x80000000

 

# MMIO - Device registers

MmioBase=0xF0000000

MmioSize=0x10000000

 

# PCI - Device BARs (use "auto" for dynamic placement)

PciMemBase=auto

PciMemSize=0x100000000

 

[System]

MemorySizeGB=32

 

 

RESULT:

  With 32 GB RAM → PCI at 0x880000000 (34 GB)

  With 64 GB RAM → PCI at 0x1080000000 (66 GB)

  With 4 GB RAM  → PCI at 0x180000000 (6 GB)

 

 

END OF VISUAL MAP
