CBox

 

 

Rules.

•Owns Branch Prediction and Branch Resolution

•Is truth source for prediction + correctness checking.

•Pipeline is truth source for execution + redirection.

•No global registration needed.

•Only reacts to IPR writes

•Never executes instructions.

 

Interfaces.

•Prediction Interface

bool predictBranch_IBox(quint64 pc, qint32 disp);

quint64 getPredictedBranchTarget_IBox(quint64 pc, qint32 disp);

 

•MisPrediction Handling (stage_EX)

void handleBranchMisprediction_EBox(quint64 correctedPC);

 

Responsibilities.

 

| Area                     | Function                                   |

| ------------------------ | ------------------------------------------ |

| **Prediction**           | compute predicted taken & predicted PC     |

| **Training**             | update BranchPredictor after actual result |

| **Mispredict detection** | predicted vs actual                        |

| **Mispredict signal**    | return `mispredict=true` to pipeline       |

| **No PC writes**         | CBox is NOT allowed to write PC directly   |
