# ✅ **System Side-Effect Checklist (SSEC)**

 

## **Facility:** `EV6_Hooks`

 

**Purpose:**

 

EV6_Hooks implements the architecture-specific behavior for EV6 IPRs (Internal Processor Registers).

These hooks run during:

 

* PALcode IPR writes/reads

* AST behavior

* IPL promotion/demotion

* Exception summary updates

* Translation subsystem interactions

* Interrupt eligibility changes

* Mode transitions (CM / PS)

* TLB behavior (via MM_STAT, VA, EXC_ADDR etc.)

 

This module does **not own state**; rather, it coordinates updates to:

 

* CPUStateIPRInterface (CSII)

* IPRBank

* HWPCB Bank

* IRQController

* ReservationManager

* EV6 TLB/PTE subsystem

* AlphaCPU (for exception, trap, interrupt pending state changes)
