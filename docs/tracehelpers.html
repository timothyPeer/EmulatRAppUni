Notes (non-table, for clarity)

Design intent (confirmed by structure):

•QString overloads are the canonical API

•const char* resolves string literal ambiguity

•std::string overloads exist only when explicitly used

All functions are:

•AXP_ALWAYS_INLINE

•Zero-overhead when logging is disabled

•Safe across hot paths (instruction retire, traps, MMU, PAL)
