# EmulatR Initialization Architecture

 

**Version:** 1.0  

**Date:** December 2025  

**Pattern:** QScopedPointer + Meyer's Singleton

 

---

 

## Overview

 

The EmulatR initialization subsystem uses a **hidden initialization pattern** where:

 

1. **QScopedPointer** manages object lifetime (automatic cleanup)

2. **Meyer's Singleton** provides global access

3. **main.cpp** only tracks initialization, doesn't perform it

 

This provides:

- ✅ Clean separation of concerns

- ✅ Automatic resource cleanup

- ✅ Single point of initialization

- ✅ Easy testing and mock injection

## Key Components

 

### 1. **EmulatR_init** (Hidden Initialization)

 

**Location:** `EmulatR_init.h`

 

**Responsibilities:**

- Load configuration from INI file

- Create and own subsystems via QScopedPointer

- Initialize subsystems in correct order

- Wire subsystems to SMPManager

- Provide accessors for global_*() functions

- Automatic cleanup on destruction

 

**Key Methods:**

```cpp

bool initializeSystem(const QString& configPath);

void initializeDeviceInterrupts();

void initializeDevices();

void shutdown();

 

IRQController* irqController() const;

SafeMemory* safeMemory() const;

GuestMemory* guestMemory() const;

```

 

---

 

### 2. **QScopedPointer Members**

 

**Owned by EmulatR_init:**

 

```cpp

QScopedPointer<IRQController> m_irqController;

QScopedPointer<SafeMemory> m_safeMemory;

QScopedPointer<GM_ALPHA::GuestMemory> m_guestMemory;

QScopedPointer<PlatformAddressMap> m_platformAddressMap;

QScopedPointer<IDeviceEmulator> m_deviceEmulator;

```

 

**Benefits:**

- Automatic deletion when EmulatR_init is destroyed

- No manual delete required

- Exception-safe (RAII pattern)

- Clear ownership semantics

 

---

 

### 3. **Global Accessor Wiring**

 

**Pattern:**

 

```cpp

// global_IRQController.h

inline IRQController& global_IRQController() noexcept {

   IRQController* irqCtrl = global_EmulatR_init().irqController();

   Q_ASSERT(irqCtrl);  // Ensure initialized

   return *irqCtrl;

}

```

 

**Flow:**

1. User calls `global_IRQController()`

2. Function calls `global_EmulatR_init()` (Meyer's Singleton)

3. Returns `m_irqController.data()` from QScopedPointer

4. Assert ensures initialization happened

## Initialization Sequence

 

### Phase 1: System Initialization

 

```cpp

auto& init = global_EmulatR_init();

bool success = init.initializeSystem("Z:/EmulatR/CONFIG");

```

 

**Steps:**

1. Load `ASAEmulatR.ini` configuration

2. Extract CPU count, memory size, system type

3. Validate configuration

4. Initialize `GlobalCPUCount`

5. Create `IRQController` (QScopedPointer)

6. Create `SafeMemory` (QScopedPointer)

7. Create `PlatformAddressMap` (QScopedPointer)

8. Create `GuestMemory` (QScopedPointer)

9. Initialize TLB system (`initializeGlobalPTE()`)

10. Initialize `ReservationManager`

11. Bind all subsystems to `SMPManager`

 

---

 

### Phase 2: Device Interrupt Registration

 

```cpp

init.initializeDeviceInterrupts();

```

 

**Registers:**

- SCSI controller → vector 0x800, IPL 20, CPU 0 (FIXED_CPU)

- Ethernet controller → vector 0x810, IPL 20 (ROUND_ROBIN)

- Serial port → vector 0x820, IPL 21, CPU 0 (FIXED_CPU)

 

---

 

### Phase 3: Device Initialization (Future)

 

```cpp

init.initializeDevices();

```

 

**Will create:**

- Device emulators (SCSI, Ethernet, Serial)

- Timer devices

- DMA controllers

## Usage Examples

 

### Example 1: Simple Initialization

 

```cpp

int main(int argc, char *argv[]) {

   QCoreApplication app(argc, argv);

   

   auto& init = global_EmulatR_init();

   if (!init.initializeSystem()) {

       return 1;

   }

   init.initializeDeviceInterrupts();

   

   return app.exec();

}

```

 

---

 

### Example 2: Using Global Accessors

 

```cpp

// After initialization, anywhere in code:

 

void someFunction() {

   // Access IRQ controller

   auto& irqCtrl = global_IRQController();

   irqCtrl.assertIRQ(0x800);  // SCSI interrupt

   

   // Access memory

   auto& safeMem = global_SafeMemory();

   safeMem.write64(0x1000, 0xDEADBEEF);

   

   // Access guest memory

   auto& guestMem = global_GuestMemory();

   MEM_STATUS status = guestMem.read64(0x2000, data);

}

```

 

---

 

### Example 3: Custom Configuration Path

 

```cpp

int main(int argc, char *argv[]) {

   QCoreApplication app(argc, argv);

   

   // Parse command line args

   QString configPath = argc > 1

       ? QString(argv[1])

       : "Z:/EmulatR/CONFIG";

   

   auto& init = global_EmulatR_init();

   if (!init.initializeSystem(configPath)) {

       qCritical() << "Failed to initialize with config:" << configPath;

       return 1;

   }

   

   init.initializeDeviceInterrupts();

   

   return app.exec();

}
```

## Migration from Old main.cpp

 

**Old main.cpp (manual initialization):**

```cpp

int main(int argc, char *argv[]) {

   QCoreApplication app(argc, argv);

   

   // Manual initialization (50+ lines)

   auto& config = global_EmulatorSettingsInline();

   config.loadFromIni("Z:/EmulatR/CONFIG/ASAEmulatR.ini");

   

   quint8 cpuCount = config.data.system.processorCount;

   quint64 ramSize = config.data.system.memorySizeGB * GB;

   

   global_SafeMemory().initialize(ramSize);

   GlobalCPUCount::initialize(cpuCount);

   global_ReservationManager().initialize();

   

   auto& smpMgr = SMPManager::instance();

   smpMgr.setCpuCount(cpuCount);

   smpMgr.bindSafeMemory(&global_SafeMemory());

   smpMgr.bindIRQController(&global_IRQController());

   // ... 20 more lines ...

   

   return app.exec();

}

```

 

**New main.cpp (clean tracking):**

```cpp

int main(int argc, char *argv[]) {

   QCoreApplication app(argc, argv);

   

   auto& init = global_EmulatR_init();

   if (!init.initializeSystem()) return 1;

   init.initializeDeviceInterrupts();

   

   return app.exec();

}

```

 

**Reduction:** 50+ lines → 6 lines ✅

## Troubleshooting

 

### Problem: Assertion failure in global_IRQController()

 

```

Q_ASSERT(irqCtrl);  // Fails!

```

 

**Cause:** `initializeSystem()` was not called before accessing global.

 

**Solution:**

```cpp

// WRONG:

auto& irq = global_IRQController();  // No init!

 

// CORRECT:

auto& init = global_EmulatR_init();

init.initializeSystem();

auto& irq = global_IRQController();  // ✅ Works

```

 

---

 

### Problem: QScopedPointer is null

 

```cpp

SafeMemory* mem = global_EmulatR_init().safeMemory();

// mem is nullptr!

```

 

**Cause:** Initialization failed silently.

 

**Solution:** Check return value:

```cpp

if (!init.initializeSystem()) {

   qCritical() << "Initialization failed!";

   return 1;

}

```
