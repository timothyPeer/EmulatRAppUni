PAL

PAL executes in MBox

 

PALcode does things such as:

 

Change privileged registers (IPRs)

 

Manipulate the HWPCB (context block)

 

Perform TLB fills and TBIS/DTB/ITB invalidations

 

•Handle traps and exceptions

•Load/store stack frames to the process kernel stack

•Switch mode (CM bits)

•Set IPL and respond to interrupts

 

Every one of these operations requires:

 

Address translation (TLB/DTB/ITB)

•D-cache and write-buffer access

•Memory ordering semantics (MB, WMB, MB2)

•Stack pointer loads/stores

•IPR reads/writes

•Kernel mapping / PAL base lookup
