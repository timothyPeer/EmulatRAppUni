Instruction Semantics

CPU-REQ-001:

For every LL instruction: Perform load from SafeMemory; Invoke ReservationManager::reserve(cpu, pa).

CPU-REQ-002:

For every SC instruction: Query tryStoreConditional() before performing the store.

CPU-REQ-003:

CPU’s IRQ polling (hasPendingInterrupt()) must respect atomic ordering.

CPU-REQ-011:

CPU-REQ-012:

STLB/WRVPTB/PTBR changes MUST clear reservations.
